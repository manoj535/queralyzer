Instruction Manual
Description :

This application is used to format complex query access plan into easily understandable tree format. It takes access plan as input and format it in tree structure as output.


Installation :

Download this project here and render index page in any browser.


Usage :

You can provide access plan in two ways

  • Paste access plan in the text box or
  • Select a file which contains access plan.
And click on format button to see tree structure of the given access plan.


Inputs :

Below are the types of supported access plan inputs

  • CSV or text file generated on any DB client software.
    Note : While exporting access plan in CSV file,ensure that lines terminated by "\n" and fields terminated by "\t" like

  • Access plan of type


Output :

Tree structure for the given access plan.
